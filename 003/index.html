lISTAS ORDENADAS

para fazer listas ordenadas basta colocar a tag < ol > e < li > para cada item específico

ex:
    listas de compras
  1. bolacha
  2. suco tang
  3. miojo

Listas não ordenadas

basta usar a tag < ul > para listas não ordenadas


ex:
    Listas de coisas aleatórias
  • meia
  • mouse
  • pc gamer 4k brabo
  • já to sem criatividade
