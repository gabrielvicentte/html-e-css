imagens que se adaptam nos sites

basta usar a tag < picture > e dentro dela colocar a tag "img" colocando as imagens em ORDEM para ela se adaptar de forma correta. Dentro dela usar a tag < source media type > e selecionar o tamanho max da imagem com alguma folga e a outra tag < srcset > serve para colocar a prox imagem que irá aparecer após diminuir ou aumentar a página ( todas a imagens tem que possuir um nome sem espaço, caso contrário não riá funcionar) o "type" deve ser colocado no respectivo formato da imagem (img png etc) Lembrando que a tag "source media type" deve ser usada em ordem para não ocorrer um possível erro.
